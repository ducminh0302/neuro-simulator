# NEUROPREDICT

A Next.js app that recreates the supplied NEURO screens as a static experience.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Deploying to Vercel

1. Push this repository to GitHub.
2. Import the GitHub repository in Vercel.
3. Use the default Next.js framework settings.
4. Deploy without extra environment variables unless you add new features that need them.
